# CONTEXT.md

This project is a Customer Communication Dashboard.

The app manages customer messages from WhatsApp and Telegram in one dashboard.

MVP features:
- Unified inbox
- Manual reply
- Keyword auto-reply
- Message templates
- Conversation history
- MinIO attachment storage
- Redis queue
- Redis idempotency
- Redis realtime notification
- Telegram webhook
- WhatsApp webhook

The app is not an AI chatbot yet. AI Customer Service will be added after MVP is stable.
